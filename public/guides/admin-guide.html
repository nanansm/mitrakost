1. Pengenalan Role Admin

Role Admin memiliki akses hampir sama dengan Owner, kecuali menu Settings yang hanya bisa diakses Owner.

Tanggung jawab Admin: Mengelola operasional harian — approve pendaftaran, input pembayaran, input laundry, memantau komplain, dan berkoordinasi dengan penjaga.

2. Login sebagai Admin

  1. Buka halaman utama, klik Login
  2. Masukkan email dan password yang diberikan Owner
  3. Anda akan diarahkan ke dashboard admin
Jika lupa password, hubungi Owner untuk reset.

4. Workflow Harian Admin

☀️ Pagi (08.00–09.00)

  1. Login ke dashboard
  2. Cek menu Pending Approval — apakah ada pendaftaran baru semalam?
  3. Review data calon penghuni satu per satu
  4. Cek bukti transfer: sudah masuk ke rekening BCA?
  5. Jika sudah masuk → klik Approve
  6. Jika belum masuk atau data tidak lengkap → Decline dengan alasan, atau tunggu konfirmasi

🌤 Siang (12.00–13.00)

  1. Buka menu Pembayaran
  2. Cek mutasi rekening BCA — ada transfer sewa yang masuk?
  3. Klik "Tambah" untuk input pembayaran manual
  4. Isi: pilih penghuni, tipe = "Sewa", bulan, jumlah sesuai transfer, status = "Lunas"
  5. Simpan — data otomatis sync ke Google Sheets

🌅 Sore (17.00–18.00)

  1. Buka menu Komplain
  2. Lihat apakah ada komplain baru dari penghuni
  3. Update status komplain yang sudah ditangani
  4. Koordinasi dengan penjaga untuk penanganan fisik
  5. Jika ada laundry masuk hari ini → input di menu Laundry

5. Cara Input Laundry

  1. Buka menu Laundry
  2. Klik form "Input Laundry"
  3. Pilih penghuni dari dropdown
  4. Masukkan berat (kg) — sistem otomatis hitung total
  5. Klik Simpan Laundry
Data laundry otomatis masuk ke laporan pemasukan dan sync ke Google Sheets.

6. Cara Kirim Notifikasi ke Penghuni

Saat ini notifikasi otomatis dikirim via email untuk:

  • Saat pendaftaran diapprove (email berisi password login)
  • Saat pendaftaran ditolak (email berisi alasan)

Untuk komunikasi langsung, gunakan WhatsApp ke nomor penghuni yang tercatat di sistem.

7. Dos and Don'ts untuk Admin

✅ Yang Harus Dilakukan

  • Approve pendaftaran hanya setelah transfer terkonfirmasi
  • Input pembayaran segera setelah transfer masuk
  • Simpan semua bukti transfer sebelum di-approve
  • Update status komplain tepat waktu
  • Laporkan masalah teknis ke Owner

❌ Yang Tidak Boleh Dilakukan

  • Approve tanpa verifikasi bukti pembayaran
  • Share password akun admin ke orang lain
  • Menghapus data tanpa konfirmasi Owner
  • Mengubah harga kamar tanpa izin Owner
  • Mengakses Settings (hanya Owner)

8. Input Pengeluaran Operasional

Admin (dan Owner) bisa mencatat pengeluaran operasional kost di menu Pengeluaran (/dashboard/expenses). Guard tidak memiliki akses ke halaman ini.

Langkah-langkah:

  1. Buka Pengeluaran di sidebar dashboard
  2. Klik Tambah Pengeluaran
  3. Pilih Lokasi tempat pengeluaran terjadi
  4. Isi Tanggal pengeluaran
  5. Pilih Kategori yang sesuai
  6. Isi Keterangan detail
  7. Isi Jumlah dalam rupiah
  8. Klik Simpan — data otomatis sync ke Google Sheets
