2. Login Pertama

Setelah pendaftaran Anda disetujui, Anda akan menerima email berisi password login.

  1. Cek email Anda (termasuk folder Spam) — cari email dari Mitra Kost
  2. Email berisi email login dan password sementara
  3. Buka browser → akses alamat web Mitra Kost
  4. Klik Login → masukkan email dan password dari email
  5. Anda berhasil masuk! Segera ganti password
Tidak dapat email? Hubungi admin di WhatsApp +62 822-3300-5808

8. Submit Komplain

Ada AC rusak, kebocoran, atau masalah lain? Laporkan lewat aplikasi agar terdokumentasi dengan baik.

Kapan submit komplain?
Saat ada kerusakan fasilitas, masalah kebersihan, atau kejadian yang perlu ditindaklanjuti. Jangan hanya lapor via chat — submit ke aplikasi agar ada bukti dan tracking.

Cara Submit:

  1. Klik menu Komplain di navigation bar
  2. Klik Buat Komplain Baru
  3. Pilih kategori (fasilitas, kebersihan, keamanan, dll)
  4. Tulis deskripsi masalah dengan jelas
  5. Upload foto sebagai bukti (sangat disarankan)
  6. Klik Kirim

Admin akan update status komplain Anda: Pending → In Progress → Selesai

11. Aturan Kost

🕘 Jam Tenang

Jam 22.00 – 06.00 WIB adalah jam tenang. Harap jaga ketenangan, volume musik/TV dipelankan, dan tidak membuat suara berisik.

🚪 Tamu

Tamu hanya boleh berkunjung hingga jam 22.00. Tamu tidak diperkenankan menginap tanpa izin pengelola.

🗑️ Kebersihan

Jaga kebersihan kamar dan area bersama. Sampah dibuang di tempat yang disediakan. Cuci peralatan makan setelah digunakan.

🚗 Parkir

Parkir kendaraan di area yang sudah disediakan. Jangan menghalangi jalan akses darurat.

💳 Pembayaran

Bayar sewa paling lambat tanggal 5 setiap bulan. Keterlambatan lebih dari 7 hari akan dikenakan biaya tambahan.

🚫 Dilarang Keras

  • Membawa/menggunakan narkoba
  • Membawa senjata tajam/api
  • Berjudi
  • Merusak fasilitas kost dengan sengaja

Pelanggaran mengakibatkan pemutusan kontrak sewa.

Ganti Password Pertama Kali (Force Change Password)

Jika admin atau owner mereset password Anda, saat login pertama kali Anda akan diarahkan otomatis ke halaman ganti password. Anda tidak bisa mengakses dashboard sampai password diganti.

  1. Login dengan email dan password sementara yang diberikan admin
  2. Sistem otomatis redirect ke halaman Ganti Password
  3. Masukkan Password Baru (minimal 8 karakter)
  4. Ulangi password baru di kolom Konfirmasi
  5. Klik Simpan Password Baru
  6. Anda akan langsung masuk ke dashboard penghuni
Tips keamanan: Pilih password yang kuat — kombinasi huruf besar, kecil, angka, dan simbol. Jangan gunakan tanggal lahir atau nama yang mudah ditebak.

Cara Minta Reset Password ke Admin

Jika Anda lupa password dan tidak bisa login, hubungi admin untuk reset password.

  1. Hubungi admin via WhatsApp: +62 822-3300-5808
  2. Beritahu nama lengkap dan email yang terdaftar
  3. Admin akan reset password dan memberikan password sementara
  4. Login dengan password sementara → sistem akan minta Anda ganti password
⚠️ Untuk keamanan, admin tidak bisa melihat password lama Anda. Reset password akan membuat password baru yang berbeda.
